### Contributing

Your contributions are always welcome! Please read the [contribution guidelines](CONTRIBUTING.md) first.

**Guidelines for adding plugins:**

1. Make sure the plugin is related to Orca Note
2. Provide a clear description of what the plugin does
3. Include the author's name and link (if available)
4. Specify the license
5. Test that all links are working
6. Add the plugin in alphabetical order within its category
