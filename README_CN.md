### 贡献

欢迎您的贡献！请先阅读[贡献指南](CONTRIBUTING.md)。

**添加插件的指南：**

1. 确保插件与 Orca Note 相关
2. 提供插件功能的清晰描述
3. 包含作者姓名和链接（如果可用）
4. 指定许可证
5. 测试所有链接是否有效
6. 在其类别中按字母顺序添加插件
